# OpenSpec 基线规格（Base Specs）

本目录存放“稳定能力”的长期规格定义。

## 当前能力列表

- `video-ingestion`
- `transcription-pipeline`
- `sse-runtime-stream`
- `llm-runtime-config`
- `llm-summary-mindmap`
- `history-and-export`
- `web-workbench-ui`

## 维护原则

1. 变更期先更新 `docs/openspec/changes/<change-id>/specs`。
2. 功能稳定后，再同步到本目录作为基线。
3. 基线更新完成后，再归档对应 change。
